# HMS | Personal Portfolio

Welcome to the source code of my personal portfolio. This project is a curated showcase of my journey as a software engineer, photographer, and perpetual learner. Based in Taxila, Pakistan, I build digital experiences that bridge the gap between strategic logic and creative storytelling.

## 🚀 Features

- **Modern Tech Stack**: Built with semantic HTML5, modern CSS3, and modular Vanilla JavaScript.
- **Modular Components**: Reusable JS components for Navigation and Footer to ensure consistency across pages.
- **Interactive UI**:
    - **Scroll Reveal**: Elements animate into view as you scroll.
    - **Scroll Progress**: A visual indicator of your reading progress.
    - **Theme Toggle**: Dark/Light mode support.
    - **Back to Top**: Smooth navigation back to the hero section.
- **Dynamic Content**: Uses HTML `<template>` tags and JavaScript to inject project and academic data dynamically.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Built With

- **Frontend**: HTML5, CSS3, JavaScript (ES6 Modules)
- **Typography**: Cormorant Garamond (Serif), DM Sans (Sans-serif)
- **Icons**: Custom SVG icons and Feather-inspired iconography
- **Utilities**: Intersection Observer API for scroll animations
